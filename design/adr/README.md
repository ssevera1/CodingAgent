# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for CodeAgent.
Each ADR documents a significant design choice, the context that motivated it,
the alternatives considered, and the trade-offs accepted.

## Format

Each ADR follows this structure:

- **Status**: Proposed | Accepted | Deprecated | Superseded
- **Context**: The situation and forces at play
- **Decision**: What was chosen
- **Alternatives Considered**: Other options and why they were rejected
- **Consequences**: Positive outcomes and accepted trade-offs
